Support

SonicDroplet support and troubleshooting.

SonicDroplet records system audio on macOS 15 or later. If you need help, start with the notes below or open a GitHub issue.

Contact

The support channel for SonicDroplet is GitHub Issues.

Open a Support Issue

Requirements

  • macOS 15.0 or later
  • Permission to capture audio if macOS requests it
  • A save location you can write to

FAQ

Why does SonicDroplet need audio capture permission?

macOS may require permission before an app can record system audio. SonicDroplet uses that permission only to record audio.

What formats are supported?

M4A and WAV.

Where are my recordings saved?

You choose the save location each time. SonicDroplet does not auto-upload or sync your recordings.

Does SonicDroplet work offline?

Yes. Recording and export are local.

Does SonicDroplet record my screen?

No. SonicDroplet records audio only.

I granted permission but recording still does not work.

  • Confirm that your Mac is running macOS 15 or later.
  • Quit and reopen SonicDroplet.
  • Play audio through your Mac and try a short test recording.
  • Save the file locally and test playback in QuickTime.
